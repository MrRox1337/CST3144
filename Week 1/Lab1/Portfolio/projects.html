My Projects

ATMO | Autonomous Temperature MOdulator

  • Created an IoT to make a smart thermostat upgrade (add-on) using multiple microcontrollers.
  • IoT involved temperature modulation using a Lilygo T-Beam to power a servo-potentiometer duo to interface with a thermostat, multiple DHT sensors connected to Wemos D1 units that report to a ThingSpeak server periodically, and asynchronously run a Random Forest regression algorithm to create temperature predictions and feed data back to the Lilygo T-Beam.
  • Presented project at multiple events including University of Wollogong
  • Innovation Fair and received feedback nearing a business-ready project.
